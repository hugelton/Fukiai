Usage

Basic Icon Usage

Use <i> tags with class="fukiai" and name attribute (like ion-icons):

<i class="fukiai" name="control_rec_f"></i>
<i class="fukiai" name="waveform_sine"></i>
<i class="fukiai" name="symbol_speaker_f"></i>
<i class="fukiai" name="function_signal_both"></i>

CDN (Recommended)

<script src="https://cdn.jsdelivr.net/npm/fukiai@latest/dist/fukiai.js"></script>

<!-- Icons will automatically render -->
<i class="fukiai" name="control_play_f"></i>
<i class="fukiai" name="port_midi"></i>

NPM

npm install fukiai
import 'fukiai/dist/fukiai.js';
// or
require('fukiai/dist/fukiai.js');

// Use in your HTML
// <i class="fukiai" name="ui_knob"></i>

Manual Download

<link rel="stylesheet" href="fukiai.css">
<script src="fukiai.js"></script>

<!-- Then use icons normally -->
<i class="fukiai" name="waveform_square"></i>
